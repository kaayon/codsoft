# Java Internship Projects

This repository contains Java programs developed during my internship coursework.  
Each project focuses on building core programming skills such as logic building, user input handling, and working with real-world data.

---

## 📌 Projects

### 1. Grade Calculator ✅

- Accepts marks for multiple subjects  
- Calculates total marks  
- Calculates average percentage  
- Assigns grade based on percentage  

---

### 2. Currency Converter 🌐 ✅

- Takes user input for amount, base currency, and target currency  
- Builds dynamic API request URL  
- Fetches real-time exchange rates using an external API  
- Converts the amount into target currency  
- Displays the result in a formatted output  

---

### 3. ATM Interface 💳 ✅

- Menu-driven console application using `while` loop and `switch-case`  
- Allows user to:
  - Deposit money  
  - Withdraw money  
  - Check account balance  
- Implements input validation (invalid amount, insufficient balance)  
- Uses Object-Oriented Programming:
  - `Account` class for balance logic  
  - `ATM` class for user interaction  
- Demonstrates clean separation of concerns (logic vs input handling)

---

## 🛠️ Technologies Used

- Java

- External API (ExchangeRate API)
